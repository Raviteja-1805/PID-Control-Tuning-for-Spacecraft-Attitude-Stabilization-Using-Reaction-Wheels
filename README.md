Ziel:
Entwerfen und optimieren Sie einen PID-Regler für das Lageregelungssystem eines Raumfahrzeugs unter Verwendung von Reaktionsrädern. Das Ziel besteht darin, eine gewünschte Ausrichtung im Weltraum zu erreichen und aufrechtzuerhalten und dabei den Lagefehler, den Regelungsaufwand und die Einschwingzeit zu minimieren.
Verwenden Sie ein System mit vier Reaktionsrädern in Pyramidenkonfiguration mit einem Pyramidenwinkel von 50 Grad. Sie können das Reaktionsrad-System auf einer beliebigen Ebene Ihrer Wahl anordnen.



Objective: (English)
Design and tune a PID controller for a spacecraft's attitude control system using reaction wheels. The goal is to achieve and maintain a desired orientation in space while minimizing attitude error, control effort, and settling time.
Use a 4 reaction wheel system in the pyramidal configuration with pyramidal angle - 50 deg. You can place the reaction wheel system along any plane of your choosing.
